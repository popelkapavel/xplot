XPlot for Windows

Xplot is great plotting tool for creating and viewing 2d/3d graphs running on unix/X-window. The original program can be found at ftp.ma.utexas.edu/pub/mzou . Here are only some comments to Windows version of this tool.

Windows version

The possibility of puting files with formulas on the command line, so you can write formulas in file with any text editor and then use xplot to see results. Also you can switch off calculating of polygon intersections, which can consume a lot of time and on some graphs is not nececery, but couse some mismatch in visibility.
To compile Windows version mingw/GRX is needed.
